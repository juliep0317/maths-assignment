Maths Assignment

For my maths assignment project, I was tasked to create a calculation using Javascript that calculated the mean of 5 decimal numbers.

I started by creating a array for the provided decimals numbers, then I created a function for the sum values.

After the array and function was created, I created a for loop and used a Math object to finally calculate the mean.

In order to calculate the mean, all the numbers need to be added together and then divided by the amount of number.

The decimals numbers used in this assignment were 4.2, 5.1, 3.8, 4.7, 5.3.

Press F12 to open up the developer tools, that will show the result in the console.
